## What I do

I build and deploy machine learning systems for industrial operations. At BP, my work includes:

- **Refinery Blending Optimization** — real-time blend-quality prediction at BP's Whiting and
  Rotterdam refineries; a Bayesian model, **A/B-tested to production** via a model registry,
  supporting an estimated **$5–7M/yr** revenue impact
- **ALI-C — Model Monitoring** — production model-monitoring system using statistical drift
  detection (CUSUM, EWMA, Hotelling's T², Q/SPE), with A/B testing of candidate models
- **Sand Detection & Alarm Optimization** — a general, asset-agnostic method that cuts
  **alarms 10–90% per well**, offline-tested on subsea wells and risers at Thunder Horse
  (Gulf of Mexico) and Trinidad

*Built as self-hosted Docker apps, integrated with Palantir Foundry (Ontology + API).*
